# 🤖 Nexus AI - Neural Link Beta Program

This is a professional business one-pager designed for the **Nexus AI** recruitment phase. The project focuses on advanced HTML forms, responsive design, and data handling using HTTPBin.

## 🎯 Project Goals 
* **Responsive Hero Section:** Featuring a high-tech background video with a glassmorphism overlay. 
* **Advanced Beta Signup Form:** Data submission to 'https://httpbin.org/anything' using 'POST' method. 
  -Multiple input types: Text, Email, Password, Radio buttons, and Checkboxes.
**Modern UI:** A sleek 'Tech-Purple' aesthetic using CSS variables and modern layout techniques. 
**Fully Responsive:** Optimized for devices from 320px to 1600px using fluid typography (clamp).

## 🛠️ Built With 
* **HTML5:** Semantic structure for SEO and better accessibility. 
* **CSS3:** Custom properties (variables), Flexbox, and CSS Grid for layout. 
* **Modern CSS:** Features like backdrop-filter for glassmorphism and object-fit for video backgrounds. 
* **Git/GitHub:** Version control and deployment. 
* **Cloudflare Pages:** High-performance hosting with CI/CD.

## 💡 Learning Reflections
* **Form Logic:** I learned how to connect labels and inputs properly to ensure high accessibility and a better user experience.
* **Accessibility:** It's not just about the looks! I made sure all inputs have proper labels so that the site is usable for everyone, which is something I'll carry with me into every future project.  
* **Visual Hierarchy:** Balancing a heavy element like a video background with clear, readable call-to-action buttons. 
* **Layout Mastery:** Combining negative margins with z-index to create a layered 'floating' effect for the form over the hero section. 
* **Problem Solving:** Working through the challenges of anchor links and positioning when using overlapping layouts. I had a real 'aha!' moment when I figured out how to fix the Hero button.
* **The Details Matter:** Small things like adding a custom Favicon and a styled footer with my name made the project feel complete and professional.
